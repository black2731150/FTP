# FTP Server

# 项目结构
```sh
.
├── bootstrap           #初始化函数
├── client              #客户端上传文件工具
├── common              #公用函数
├── config              #配置信息函数
├── config.yaml         #配置文件
├── database            #数据库操作
├── files               #存放上传的文件
├── global              #存放全局变量
├── main.go             #主函数
├── routes              #路由管理
├── web                 #前端项目
├── go.mod              
├── go.sum
```
